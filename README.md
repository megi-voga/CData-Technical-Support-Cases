# CData Technical Support Cases

This repository contains Python implementations developed for four IPWorks technical support scenarios.

## Cases

- Case 1 – TCP Client/Server Communication
- Case 2 – Extracting a Single File from a ZIP Archive
- Case 3 – SFTP Host Key Verification
- Case 4 – Reading Product Titles from a JSON API

Each case includes:

- Python source code
- Execution instructions
- Validation screenshots
- Individual README documentation

The complete analysis, research process, recommended solution, and customer responses are documented separately in the accompanying report.
